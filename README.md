# Suhas Memory Pad

A playful tablet-first remembering app for a 6-year-old. It runs as a web app, saves memories in the browser, and includes animated graphics, voice playback, optional voice input, optional browser notifications, and an offline-ready service worker.

## Run

```bash
python3 -m http.server 4174 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:4174
```

## What Is Included

- Big add-memory composer with large touch targets.
- Kid-friendly reminder choices: Today, Tomorrow, Bedtime, When I Ask.
- Animated mascot, clouds, stars, rocket effects, and confetti.
- Stable memory cards with large speak, done, and delete buttons.
- Ask Buddy recall drawer for finding saved memories.
- Local browser storage so memories stay after refresh.
- Optional speech playback, voice input, and notifications when supported.
